# SpNp

Social and Professional Networking Platform

# Key features

-  User signup and login
-  Create and manage user profiles
   -  Full name, address and birthday
   -  Bio
   -  Work experience
      -  Start and end dates
      -  Position and company
      -  Job description
   -  Education
   -  Skills
   -  Languages
-  Manage posts
-  Manage comments
-  Upload avatars
